# Tech Stack - São Luís Garagem

## Core Framework & Language
- **Next.js 16 (App Router):** The foundation of the web application, providing server-side rendering, routing, and optimized performance.
- **TypeScript:** Ensuring type safety and better developer experience across the codebase.
- **React 19:** Utilizing the latest React features for building a dynamic and responsive UI.

## Styling & UI
- **Tailwind CSS v4:** Modern utility-first CSS framework for fast and responsive styling, using the latest v4 features.
- **Framer Motion:** For smooth and engaging animations and transitions.
- **Lucide React:** A consistent and professional set of icons.

## Backend & Integration
- **Server Actions:** Handling form submissions and server-side logic within Next.js.
- **Nodemailer:** Enabling functional email dispatch for the contact form.
- **Zod:** Robust schema validation for form data and internal data structures.

## Development Tools
- **ESLint:** Enforcing code quality and consistency.
- **PostCSS:** For processing CSS with Tailwind.
